# Sum of Two Numbers - 1

Write a code to print sum of two numbers. One of the number is 10 and the other is 20. 

**Sample Test Case Explanation**
```
Input:
No input

Output:
30

Explanation:
a = 10, b = 20
sum = 10 + 20 = 30
```
