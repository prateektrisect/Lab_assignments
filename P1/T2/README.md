# Sum of Two Numbers - 2
Given two integers as input. Print the sum of these integers. 

**Sample Test Case**
```
In the input format, two numbers as an integer where first value is given 10 and second is 20. 
Input:
10 20

Output:
30

Explanation:
a = 10, b = 20
sum = 10 + 20 = 30
```
